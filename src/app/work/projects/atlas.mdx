## Aperçu

Atlas est une plateforme permettant de découvrir les films et séries TV en utilisant l'API TMDB.

## Caractéristiques

- Interface utilisateur intuitive pour explorer les films et séries
- Intégration avec l'API TMDB pour accéder à une vaste base de données de médias
- Développé avec React et Next.js pour une expérience utilisateur optimale
- Styling avec TailwindCSS pour un design moderne et responsive

## Technologies utilisées

- **React**: Pour construire l'interface utilisateur interactive
- **Node.js**: Pour le backend et la gestion des API
- **Next.js**: Pour le rendering côté serveur et l'optimisation des performances
- **TailwindCSS**: Pour un styling efficace et cohérent
