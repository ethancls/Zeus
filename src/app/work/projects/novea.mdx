# Novea - Plateforme d'Intérim

## Aperçu du projet

Novea est une plateforme d'intérim complète développée entièrement from scratch. Elle facilite la mise en relation des entreprises cherchant des travailleurs temporaires et candidats disponibles pour des missions d'intérim, avec une gestion complète du processus de recrutement et de mission.

## Fonctionnalités principales

### Pour les entreprises
- **Création d'offres** : Publication de missions d'intérim détaillées
- **Gestion des candidatures** : Interface de tri et sélection des candidats
- **Planning des missions** : Calendrier intégré pour la gestion temporelle
- **Tableau de bord** : Analytics et métriques de performance

### Pour les candidats
- **Profil personnalisé** : CV + Lettre de motivation
- **Recherche d'offres** : Filtres avancés par localisation, domaine, salaire
- **Candidatures en un clic** : Processus de candidature simplifié
- **Suivi des missions** : Historique et planning personnel
- **Système de notation** : Évaluations mutuelles entreprise/candidat
- **Notifications temps réel** : Alertes pour nouvelles offres et réponses

## Architecture technique

### Frontend
- **TypeScript** : Typage statique pour la robustesse
- **Vite** : Build tool moderne pour un développement rapide
- **Vue.js** : Framework JavaScript pour une interface utilisateur réactive
- **TailwindCSS** : Framework CSS

### Backend
- **Node.js** : Runtime JavaScript côté serveur
- **Prisma ORM** : Interface type-safe avec la base de données
- **PostgreSQL** : Base de données relationnelle robuste
- **JWT** : Authentification sécurisée par tokens

## Défis techniques relevés

### Architecture scalable
- Conception d'une architecture modulaire pour supporter la croissance
- Optimisation des requêtes base de données
- Mise en place d'un système de cache efficace

### UX/UI complexe
- Interface adaptée aux différents types d'utilisateurs
- Design responsive pour tous les appareils

### Conformité légale
- RGPD et protection des données personnelles

### Performance et sécurité
- Optimisation pour gérer des milliers d'utilisateurs simultanés
- Sécurisation des données sensibles (salaires, informations personnelles)

## Modules développés

### Gestion des utilisateurs
- Authentification multi-rôles (candidat, administrateur)
- Authentification à deux facteurs (2FA) via email

### Moteur de recherche
- Recherche full-text optimisée
- Filtres avancés et sauvegarde de recherches
- Géolocalisation pour missions locales

Novea représente un projet entrepreneurial complet, de la conception à la mise en production, illustrant ma capacité à développer des solutions business.
